    1. Did This can be difined as Doctor id it is auto increment

    1. Cid This can be difined as clinic id it is auto increment

    2. Name : is the name of clinic.

  • Appointment table Attributes

    1. Aid This can be difined as Appointment id it is auto increment

  • Service table Attributes

    4. Pid :is the foreign key from patient table.
